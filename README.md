# Typing-Speed-Computer
Speed typing game for 386 computer in mwc.

## Dependencies
MSCLOADER by piotrulos
I386API by tommojphillips

## Building from Source

Compile `computer386.cs` targeting .NET 3.5. with references to `MSCLoader.dll`, `I386API.dll`, and `UnityEngine.dll`.
